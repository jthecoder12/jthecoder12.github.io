Brick Breaker

Brick Breaker is a recreation of Atari Breakout written in Java and libGDX.

I plan to add multiplayer and AI to the game.

This game is based on the original Brick Breaker that I made in Unity.

There are downloads available for Windows, MacOS (both Intel and Silicon), and Linux. I have a web version but it is limited as I had to minimize it for TeaVM support. There will be no multiplayer and there is no ImGui menu.
